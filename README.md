# **PHB God‑Code Vortex**  
### *A distributed, evolving symbolic engine with real‑time P2P synchronisation*

## 📌 Overview
PHB God‑Code Vortex is a modular symbolic engine producing Codex pages, voxel layers, veil states, energy signatures, portals, and full JSON evolution packets.  
A P2P layer synchronises evolution across devices using `network_state.json`.

## 🚀 Installation
Run:
\`\`\`
bash installer.sh
\`\`\`

## ▶️ Running
\`\`\`
./gcs/run_phb_p2p.sh
\`\`\`

## 🌐 Multi‑Device Evolution
All peers broadcast and receive JSON evolution packets, forming a distributed symbolic mesh.
